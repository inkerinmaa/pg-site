Как это работает

Ваши знания становятся обширнее с каждым днем

01

Выберите программу обучения

Обучение языкам программирования? У нас есть курсы по Scratch для детей, для детей 7-16 лет есть курсы по Python, HTML + CSS + Javascript, а также C#. Может хотите заниматься математикой или английским языком? Есть программы и по этим направлениям

02

Выберите желаемый темп занятий

У нас есть возможность выбрать комфортную для вас скорость обучения. Можно заниматься в группе с фиксированным графиком - 2 занятия в неделю, можно выбрать индивидуальный план с преподавателем, а так же есть возможность приобрести курс и заниматься самостоятельно в своем темпе, отправляя домашние задания преподавателю на проверку

03

Закрепляйте свои знания

Лучшим доказательством эффективности обучения будут отличные оценки в школе, безупречно написанные контрольные работы и выполненные вами проекты. На каждом нашем курсе мы подробно разбираем методические материалы, добиваемся полного понимания предмета учеником. На курсах по программированию каждый ученик в итоге сделает свой проект
